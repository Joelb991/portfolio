Integer Programming for UI Layout

Spotify UI Optimization

Developed a mixed-integer optimization model using Python and Gurobi to determine the optimal placement, sizing, and page assignment of UI modules across Spotify's mobile app interface.

Python Gurobi Optimization Tkinter

Mathematical Formulation

The optimizer reads an Excel workbook with four sheets — user group preference scores, module max sizes, page weights, and allowable percentage sizes — and solves a binary integer program to determine the optimal layout.

Data & Parameters

$I$ Set of UI modules (e.g. search, favorites, new release)
$J$ Set of user segments
$K$ Set of percentage sizes
$P$ Set of pages
$s_{ij}$ Preference score of module $i$ for user group $j$
$w_p$ Weight assigned to page $p$
$l_i$ Maximum allowed percentage for module $i$
$M_{min}$ Minimum modules per page
$M_{max}$ Maximum modules per page

Decision Variable

Binary Variable

$$Z_{ikp} = \begin{cases} 1 & \text{if module } i \text{ is assigned size } k \text{ on page } p \\ 0 & \text{otherwise} \end{cases}$$

Objective

Maximize Weighted User Preference

$$\text{Maximize} \quad \sum_{i \in I} \sum_{j \in J} \sum_{k \in K} \sum_{p \in P} s_{ij} \cdot w_p \cdot Z_{ikp}$$

Constraints

1. Each module can appear on at most one page
$$\sum_{k \in K} \sum_{p \in P} Z_{ikp} \leq 1 \quad \forall \; i \in I$$
2. Module cannot exceed its maximum allowed size
$$Z_{ikp} = 0 \quad \text{for all } k > l_i$$
3. Total sizes on each page must not exceed 100%
$$\sum_{i \in I} \sum_{k \in K} k \cdot Z_{ikp} \leq 1.0 \quad \forall \; p \in P$$
4. Recommendation module must be placed on Page 1
$$\sum_{k \in K} Z_{R,k,1} = 1$$
5. Min/max number of modules per page
$$M_{min} \leq \sum_{i \in I} \sum_{k \in K} Z_{ikp} \leq M_{max} \quad \forall \; p \in P$$
